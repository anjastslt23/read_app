## Getting Started

1. **Clone this repository:**
    ```bash
    git clone https://github.com/anjastslt23/reading-app.git
    cd reader-app
    ```

2. **Install dependencies:**
    ```bash
    flutter pub get
    ```

3. **Run the app:**
    ```bash
    flutter run
    ```
